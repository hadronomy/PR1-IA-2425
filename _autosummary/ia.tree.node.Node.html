ia.tree.node.Node#

class ia.tree.node.Node(name: str = '', separator: str = '/', compare_by: str = 'name', **kargs: any)#

Bases: BaseNode

Node is an extension of BaseNode, and is able to extend any Python class.

Nodes can have attributes if they are initialized Node, dictionary. Nodes can be linked to each other with parent and children setter methods.
